# Knowledge Base Spec

## Status

Phase 2 implemented.

## Scope

The Knowledge Base stores editable user profile information and reusable application answers. Phase
2 includes CRUD, local persistence, section grouping, sorting, validation, and simple keyword
search.

## Sections

- Personal
- Professional
- Education
- Experience
- Links
- Work Authorization
- Behavioral Answers
- Company Specific Answers
- Miscellaneous

## Non-Goals

No AI rewriting, embeddings, semantic search, vector database, job search, automation,
authentication, or cloud sync is included in Phase 2.
